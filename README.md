# TikTok Creator Project

This project enables users to create TikTok-style videos using AI and customization tools.

## Backend
### Requirements
- Python 3.8+
- FastAPI
- MongoDB

### Setup
1. Navigate to the backend directory: `cd backend`
2. Create a virtual environment: `python3 -m venv venv`
3. Activate it: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Start the server: `uvicorn app:app --reload`

## Frontend
### Setup
1. Navigate to the frontend directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`

### Features
- Script generation using AI.
- Video rendering and customization.
- Publishing to TikTok or saving locally.
